## 2024-03-24 - Extracting Flutter UI helpers into const StatelessWidgets
**Learning:** In Flutter, using large helper methods like `_buildToolCard` or `_buildFileItem` that return widgets causes the framework's element tree diffing algorithm to perform suboptimally. When the parent widget rebuilds, these methods execute unconditionally and create entirely new widget instances.
**Action:** Extract local UI helper methods that return widgets into explicit `const StatelessWidget` classes. This improves widget diffing efficiency, better scopes rebuilds, and allows the framework engine to short-circuit the widget building process entirely when the constructor arguments are constants. Remember to replace map color lookups like `Colors.purple[50]` with explicit hex literals like `Color(0xFFF3E5F5)` to enable `const` instantiation.
