## 2024-06-25 - Flutter Performance Best Practices
**Learning:** In modern Flutter (3.27+), `Color.withOpacity()` is deprecated; use `Color.withValues(alpha: ...)` instead to properly handle color spaces, avoid precision loss, and resolve analyzer warnings. Also, extracting local UI helper methods (like `_buildToolCard` or `_buildFileItem`) that return Widgets into explicit `const StatelessWidget` classes is a recommended Flutter performance practice. It helps the framework's element tree diffing algorithm run more efficiently, better scopes rebuilds, and allows the engine to short-circuit the widget building process.
**Action:** Always prefer `Color.withValues(alpha: ...)` over `Color.withOpacity()`. Automatically refactor `Widget _buildSomething(...)` methods into `const StatelessWidget` classes when optimizing UI code, and make sure to instantiate them with the `const` keyword where appropriate.
