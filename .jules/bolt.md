## 2024-04-07 - Optimizing Flutter UI Rebuilds via Const Extraction
**Learning:** In Flutter, using helper methods (like `_buildToolCard` or `_buildFileItem`) inside a large `StatelessWidget` does not prevent them from executing whenever the parent widget rebuilds. By extracting these helper methods into separate, dedicated `StatelessWidget` classes and invoking them with the `const` keyword, Flutter can completely skip the build phase for these sub-trees during parent rebuilds, providing a significant and safe performance improvement.
**Action:** When identifying large monolithic screens with repetitive or static sub-components generated by helper methods, extract those components into private `StatelessWidget` classes and ensure they are instantiated with `const` to minimize the widget rebuild footprint.
